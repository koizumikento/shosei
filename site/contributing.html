Contributing

このリポジトリは spec-first です。挙動変更を伴う提案では `docs/specs/` と `docs/adr/` を同じ change で見直します。

Before opening a PR

  • CLI 名は `shosei` を使う
  • config 名は `book.yml` と `series.yml` を維持する
  • repo model を壊す変更では spec と ADR を先に確認する
  • source of truth は `docs/specs/` と `docs/adr/`

Validation

Rust 側の変更を検証するときは、次のコマンドをそのまま使います。

cargo fmt
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
cargo test -p shosei-core --test repo_discovery
cargo run -p shosei-cli --bin shosei -- --help
