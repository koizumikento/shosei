Config Reference

`shosei init` が生成する `book.yml` / `series.yml` / `books/<book-id>/book.yml` の意味を、普段触る項目に絞ってまとめます。

Markdown version

どのファイルを見るか

single-book では root の book.yml を編集します。

series では、シリーズ共通項目は root の series.yml、巻固有の項目は books/<book-id>/book.yml を編集します。

single-book の基本形

project:
  type: novel
  vcs: git
  version: 1
book:
  title: "My Book"
  authors:
    - "Ken"
  language: ja
  profile: novel
  writing_mode: vertical-rl
  reading_direction: rtl
layout:
  binding: right
  chapter_start_page: odd
  allow_blank_pages: true
manuscript:
  chapters:
    - manuscript/01-chapter-1.md
outputs:
  kindle:
    enabled: true
    target: kindle-ja
validation:
  strict: true
  epubcheck: true
  kindle_previewer: false
  accessibility: warn
git:
  lfs: true
editorial:
  style: editorial/style.yml
  claims: editorial/claims.yml
  figures: editorial/figures.yml
  freshness: editorial/freshness.yml

series の役割分担

series.yml はシリーズ共通設定、books/<book-id>/book.yml は巻ごとの上書きです。

series:
  id: my-series
  title: "My Series"
  language: ja
  type: novel
shared:
  assets:
    - shared/assets
  styles:
    - shared/styles
  fonts:
    - shared/fonts
  metadata:
    - shared/metadata
defaults:
  book:
    profile: novel
    writing_mode: vertical-rl
    reading_direction: rtl
  layout:
    binding: right
    chapter_start_page: odd
    allow_blank_pages: true
books:
  - id: vol-01
    path: books/vol-01
    number: 1
    title: "Volume 1"
project:
  type: novel
  vcs: git
  version: 1
book:
  title: "Volume 1"
  authors:
    - "Ken"
  language: ja
layout:
  binding: right
  chapter_start_page: odd
  allow_blank_pages: true
manuscript:
  chapters:
    - books/vol-01/manuscript/01-chapter-1.md
editorial:
  style: books/vol-01/editorial/style.yml
  claims: books/vol-01/editorial/claims.yml
  figures: books/vol-01/editorial/figures.yml
  freshness: books/vol-01/editorial/freshness.yml

よく触る項目

Field どこにあるか 意味
project.typebook.ymlプロジェクト種別
book.titlebook.yml, books/<book-id>/book.ymlその本のタイトル
book.authors同上著者一覧
book.languageどちらにもありうる本文の言語コード
book.profile主に book.yml か series.yml.defaults.book出力 preset
book.writing_mode同上horizontal-ltr か vertical-rl
book.reading_direction同上ltr か rtl
layout.bindingbook.yml または series.yml.defaults.layout綴じ方向
layout.chapter_start_page同上章を奇数ページ始まりにするか
layout.allow_blank_pages同上空ページを入れてよいか
manuscript.chaptersprose の book.yml章順そのもの
outputs.kindle.*book.yml または series.yml.defaults.outputsKindle 出力設定
outputs.print.*同上print 出力設定
pdf.engineprose の print 時print PDF のレンダラ
print.trim_sizeprint 時仕上がりサイズ
editorial.*prose の book.ymleditorial sidecar file への参照

validation.epubcheckbook.yml または series.ymlKindle/EPUB 出力で epubcheck を使うか
validation.kindle_previewerbook.yml または series.ymlKindle Previewer の conversion check を opt-in するか
git.lfsbook.yml または series.ymlGit LFS 前提か

series.yml でよく触る項目

Field 意味
series.idシリーズ識別子
series.titleシリーズ名
shared.assetsshared cover や画像の探索先
shared.stylesshared CSS の探索先
shared.fontsshared font の探索先
shared.metadataseries metadata の置き場
defaults.book.*各巻の既定値
defaults.layout.*各巻の layout 既定値
books[].id--book で指定する巻 ID
books[].path巻ディレクトリの repo-relative path
books[].number巻番号
books[].title巻タイトル

manga の項目

project.type: manga のときは prose の manuscript / editorial ではなく manga block を使います。

Field 意味
manga.reading_directionページ読み順
manga.default_page_side本文の最初のページを左右どちらに置くか
manga.spread_policy_for_kindle見開きを Kindle 向けにどう落とすか
manga.front_color_pages巻頭カラーの枚数
manga.body_mode本文ページの既定

注意点

  • config の path は repo-relative
  • path 区切りは /
  • prose の章順は manuscript.chapters の配列順で決まります
  • series では series.yml の defaults が先に入り、巻固有 book.yml が上書きします
  • 詳しい制約や許容値は schema を参照してください
